Companies

AOC international

AOC international is a Taiwanese electronics company. They primarily work with AI dev tools for their assembly line and with planning and designing tools.
The main IT roles in their company are:

  • Project manager
  • Designer
  • Data scientist
  • Quality assurance analyst
  • Hardware engineer

OnePlus Technology

OnePlus Technology is a Chinese smartphone manufacturer. They primarily work with c++, java, AI dev tools and some planning and designing tools.
The main IT roles in their company are:

  • Project manager
  • Data scientist
  • Software developer
  • Security specialist
  • Designer

Adobe Inc.

Adobe Inc. is an American computer software company. They primarily work with pascal, c++ and java.
The main IT roles in their company are:

  • Data scientist
  • Software developer
  • Application support
  • UX designer
  • Project manager

Rockstar North

Rockstar North is a British video game development company. They primarily work with c++, c# and PHP
The main IT roles in their company are:

  • Project manager
  • Software developer
  • UX designer
  • Data scientist
  • Networking engineer / infrastructure specialist (for online games)

Tencent Ltd

Tencent Ltd is a Chinese company that makes various internet-related products. They are a very big company with over 60 000 employees so they work with a lot of tech. Their gaming branch mostly uses c++ while their social media branch is likely more about PHP and javascript
The main IT roles in their company are:

  • Project manager
  • Software engineer
  • Web developer
  • Data scientist
  • UX designer

Macaw

Macaw is a dutch IT company that helps companies turn their concepts into reality. They help make people successful. They work in a waterfall/agile way and they use a lot of new software that microsoft develops.
The main IT roles in their company are:

  • Data consultant
  • Data scientist
  • Web developer
  • Software engineer
  • Project manager

Yoursuprse

Yoursuprise is a dutch company that sells personalized gifts. They work in a very agile way and they use a range of languages including html and javascript.
The main IT roles in their company are:

  • Data consultant
  • Data scientist
  • Web developer
  • Software engineer
  • Project manager
